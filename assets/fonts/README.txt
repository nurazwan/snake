FONTS FOLDER
============

Put a pixel font file here named "pixel.ttf" for the retro look.

If you don't have a font, the game will use your system's default font.

FREE PIXEL FONTS:
- https://www.dafont.com/bitmap.php
- https://fonts.google.com/?category=Monospace (look for "Press Start 2P")

Just download a .ttf file and rename it to "pixel.ttf"
